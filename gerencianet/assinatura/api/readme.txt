SDK PHP Gerencianet Pagamentos - Arquivo .zip - Para utilização da API Gerencianet Pagamentos do Brasil

Os arquivos aqui presentes são exatamente o que são baixados quando utilizado o Composer para controle de dependências (veja em https://dev.gerencianet.com.br/docs/instalacao-da-api).

A pasta "api", presente neste .zip é necessária quando você opta por não utilizar o Composer para baixar as dependências da SDK em PHP da Gerencianet.

Para prosseguir com a utilização, você deve copiá-la para o diretório raiz de sua aplicação/site/sistema e referenciar o arquivo "autoload.php" localizado em "api\vendor\autoload.php". Em nossos códigos de exemplos disponíveis na
documentação, essa linha com a referência deste diretório é comumente encontrada da seguinte forma: require __DIR__.'/../../vendor/autoload.php'; (veja um exemplo em https://dev.gerencianet.com.br/docs/pagar-boleto-gerar-transacao).

Importante reforçar que o conteúdo presente neste .zip é apenas um requisito para que você possa começar a utilizar a SDK em PHP da Gerencianet. Esta pasta por si só não é um "exemplo pronto" de uso da API Gerencianet, mas a SDK em PHP da Gerencianet que permite a utilização da API.

Como recursos de apoio, oferecemos uma documentação completa com explicações claras e objetivas, incluindo exemplos prontos (que podem ser baixados) de integrações com nossa API e vídeos em formato de aulas sobre como integrar com
a Gerencianet. Não deixe de acessar: https://dev.gerencianet.com.br/docs
